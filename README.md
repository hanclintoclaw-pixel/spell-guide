# Spell Guide

Static Shadowrun 3rd Edition spell catalogue and Sorcery/Conjuring roll aid for the Nashville Shadowrun campaign.

This first pass establishes the player-facing UI shell: spell browsing cards, casting/conjuring roll helpers, roll history, and a cyberpunk-mystic visual direction. Canonical SR3 spell data can be added after source-backed curation.

## Development

```bash
npm install
npm run lint
npm run build
```

## Deployment

The app deploys to GitHub Pages from `main` via `.github/workflows/deploy.yml`.

Live URL: <https://hanclintoclaw-pixel.github.io/spell-guide/>
